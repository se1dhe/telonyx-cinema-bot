# TELONYX CINEMA Bot

Редакционный Telegram-бот для русскоязычного канала TELONYX CINEMA.

## Что делает бот

- собирает киноновости из источников;
- отбирает только сильные инфоповоды;
- публикует посты с изображением, авторской подачей и хештегами;
- держит мягкий интервал между новостями;
- подставляет fallback-контент, если новостей нет;
- дает админу панель управления автопотоком.

Все ответы бота и публикации канала должны быть на русском.

## Setup

```bash
uv sync --extra dev
cp .env.example .env
uv run python -m telonyx_cinema_bot
```

Переменные окружения перечислены в `.env.example`.

`ADMIN_USER_IDS` можно оставить пустым для запуска без админов или указать
один/несколько Telegram user ID через запятую: `123456789,987654321`.

## Админка

Напишите боту `/start` с админского Telegram-аккаунта.

Доступные действия:

- `Редакционная очередь` — посмотреть готовые посты;
- `Автопоток` — включить публикации;
- `Пауза` — остановить автопоток на 6 часов;
- `Опубликовать сейчас` — попробовать выпустить ближайший пост;
- `Ручной пост` — скопировать готовое сообщение в канал.

## Railway

Railway использует `railway.json` и запускает worker командой:

```bash
python -m telonyx_cinema_bot
```
